# Images Folder Structure

## Aircraft Images (`/aircraft`)
Place individual aircraft photos here. Name them using the aircraft ID:
- `mb326.jpg` - Aermacchi MB-326
- `a300.jpg` - Airbus A300
- `su27.jpg` - Sukhoi Su-27
- `f16.jpg` - F-16 Fighting Falcon
- `a380.jpg` - Airbus A380
- `c130.jpg` - C-130 Hercules
- `b777.jpg` - Boeing 777
- `cessna172.jpg` - Cessna 172
- `mirage2000.jpg` - Mirage 2000
- `concorde.jpg` - Concorde
- `spitfire.jpg` - Spitfire
- `a320.jpg` - Airbus A320
- `mig29.jpg` - MiG-29
- `p51.jpg` - P-51 Mustang
- `e190.jpg` - Embraer E-190

## Category Images (`/categories`)
Place category banner images here:
- `commercial.jpg` - Commercial aviation
- `military.jpg` - Military aircraft
- `general.jpg` - General aviation
- `vintage.jpg` - Vintage aircraft

## Recommended Image Specs
- **Aircraft images**: 1200x800px, JPEG/PNG
- **Category images**: 800x600px, JPEG/PNG
- Keep file sizes under 500KB for better performance
